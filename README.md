# ChatList

Python-приложение для отправки одного промта в несколько нейросетей и сравнения ответов.

## Возможности

- Ввод нового промта или выбор сохранённого
- Параллельная отправка в активные модели
- Таблица результатов с выбором строк для сохранения
- Управление моделями (добавление, редактирование, активация)
- История сохранённых ответов с поиском
- Настройки таймаута, пути к БД и логирования
- Экспорт результатов в Markdown и JSON
- Поддержка OpenRouter, OpenAI, DeepSeek, Groq

## Требования

- Python 3.11+
- Windows / macOS / Linux

## Установка

```powershell
cd c:\Work\ChatList
pip install -r requirements.txt
Copy-Item .env.example .env
```

Укажите API-ключи в файле `.env` (или `.env.local`):

```env
OPENROUTER_API_KEY=ваш_ключ
```

При первом запуске модели с доступными ключами активируются автоматически.

## Запуск

```powershell
python main.py
```

## Использование

1. Откройте вкладку **Запрос**, введите промт и нажмите **Отправить**.
2. Отметьте нужные ответы и нажмите **Сохранить выбранные**.
3. На вкладке **Модели** добавьте или включите нейросети.
4. На вкладке **История** просматривайте сохранённые результаты.
5. На вкладке **Настройки** измените таймаут и параметры логов.

## OpenRouter

По умолчанию добавлена модель `OpenRouter GPT-4o mini`:

- URL: `https://openrouter.ai/api/v1/chat/completions`
- API ID: `openai/gpt-4o-mini`
- Переменная: `OPENROUTER_API_KEY`

Другие модели OpenRouter можно добавить на вкладке **Модели**, указав нужный `api_id` (например `anthropic/claude-3.5-sonnet`).
